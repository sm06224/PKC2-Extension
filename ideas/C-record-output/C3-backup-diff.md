# C3. backup-diff（2 PKC2 の diff）

**目的**: 2 つの container エクスポートを比較し追加/削除/変更を表示。

**UI 概要**:

- 2 つのソースセレクタ（iframe or ファイルアップロード）
- 差分ビュー: 3 列テーブル（A のみ / 変更 / B のみ）
- 色分け: 🟢追加 🔴削除 🟡変更
- エントリクリック → フィールド別 diff

**実装ノート**:

- `lid` でレコードマッチ
- `JSON.stringify(entry)` で同一判定
- 変更レコード: フィールドごとに差異ハイライト

**SR 依存**: SR-2, SR-11 | **優先度**: Tier 3
