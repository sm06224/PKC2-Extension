# C2. container-json-viewer（container 構造をツリー表示）

**目的**: container の構造をインタラクティブな折りたたみツリーで可視化。

**UI 概要**:

- 折りたたみツリー: container → entries (archetype 別) → フィールド
- archetype 別バッジ（エントリ数）
- エントリクリック → サイドパネルに詳細
- 検索 / フィルタ + 統計サマリ

**実装ノート**:

- 再帰的 DOM ツリービルダー（外部ライブラリ不要）
- 遅延レンダリング（展開時に子ノード生成）
- JSON シンタックスカラーリング（正規表現ベース）

**SR 依存**: SR-2 | **優先度**: Tier 3
