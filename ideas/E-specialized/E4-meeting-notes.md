# E4. meeting-notes（テンプレ議事録）

**目的**: 構造化テンプレートの議事録を text レコードとして送信。

**UI 概要**:

- タイトル（自動: 「会議メモ YYYY-MM-DD」）
- 参加者、議題リスト（動的追加/削除）
- 議題ごと: 議論メモ + アクションアイテム
- 決定事項、次回日程

**生成される Markdown**:

```markdown
# 会議メモ 2026-04-17
## 参加者: Alice, Bob
## 議題 1: xxx
**議論**: ...
**アクション**: - [ ] Alice: xxx (期限: 4/20)
## 決定事項
- ...
```

**実装ノート**:

- アクションアイテム → 個別 todo offer も可能（オプション）

**SR 依存**: SR-4, SR-8 | **優先度**: Tier 3
