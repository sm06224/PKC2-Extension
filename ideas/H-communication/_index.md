# H. コミュニケーション系ツール

AI / チャット型の入力・連携ツール群（2026-06-13 新路線。トラッキング正本 = GitHub Issue #110）。

PKC2 本体の「機能を足さない・削る」方針と整合させつつ、入力面（書く心理コストの最小化）と
AI 連携を拡張側で提供する。host との通信は pkc-ext チャネル（PKC2 spec §3.8）。

## ツール一覧

| # | Name | 一行サマリ | 状態 |
|---|------|----------|------|
| H1 | chat-journal | チャット型セルフメモ → 日次 textlog を propose | 出荷済み（#108 / PR #114, #115） |
| H2 | [ai-assistant](H2-ai-assistant.md) | AI チャット連携（会話ログを PKC2 へ保存） | 設計 → v1 実装中（#109） |

## 共通規律

- **データ最小化**: projection は本文・asset・revisions を含まない。実体はユーザーの送付ジェスチャ（`deliver`）でのみ届く。
- **同意モデル**: 文脈として渡す entry はユーザーが明示的に送付したものだけ。勝手に読まない。
- **受信描画は textContent / blob のみ**（`innerHTML` 禁止）。
